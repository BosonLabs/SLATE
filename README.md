# SLATE

SLATE is a unified all-in-one investment platform like Binance Launchpad and Polkastarter for Algorand, BSC, and Polkadot projects.

# Table of Contents
  - Install
  - Contribute
  - Licence

# Install:

          npm init -y
          
    # Install Hardhat:  
  
          npm install --save-dev hardhat
          npx hardhat
          npm install --save-dev @nomiclabs/hardhat-ethers ethers
          npm install --save-dev @openzeppelin/hardhat-upgrades
